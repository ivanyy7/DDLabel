# Карта проекта DDLabel

Краткий обзор структуры проекта: важные папки, ключевые файлы, точки входа.

---

## Текущая структура

- **Корень проекта** — конфигурация, документация (PLAN, MAP, LOG), README.
- **`.cursor/rules/`** — правила для чата и работы над проектом.
- **`.github/workflows/`** — CI (например, ci.yml).

Остальная структура (исходный код приложения, экраны, сервисы и т.д.) будет добавлена по мере разработки и отражена здесь.
