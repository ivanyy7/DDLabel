# Сокращения и варианты названий

## Правила

1. **Если в столбце «На этикетке» пусто** — на этикетке печатается название продукта из столбца «Продукт».
2. **Варианты названия (aliases):** один продукт можно называть по-разному (соус красный, Красный для пиццы → продукт «Красный соус»). В справочнике у каждого продукта есть список вариантов; поиск (голос/фраза) идёт по названию и по вариантам.
3. **Если программа не разобрала фразу** — в ответе выводится услышанная фраза: «Вы сказали: «…»», чтобы можно было поправиться.

## Где в коде

- **Формат записи:** `{ productName, value, unit, labelText?, aliases?: string[] }`. `labelText` — надпись на этикетке; пусто → печатается `productName`. `aliases` — варианты названия (поиск по ним даёт эту запись).
- **Хранение:** `server/shelfStorage.js` — `getByProductName(name)` ищет по `productName` и по `aliases`; возвращает запись с каноническим `productName`. `getLabelText(productName)` возвращает `labelText` или `productName`.
- **Расчёт срока:** `shelfLife.resolveExpiry` возвращает каноническое `productName` из справочника (чтобы на этикетке и в надписи использовалось одно и то же название).
- **Ошибки:** в `POST /api/parse` и `POST /api/print` при ошибке к сообщению добавляется «Вы сказали: «фраза»».
- **Интерфейс:** в модальном окне «Список продуктов» — столбцы «На этикетке» и «Варианты названия»; при пустом «На этикетке» показывается подсказка «— (название продукта)».

## Примеры

| Как можно сказать           | Продукт      | На этикетке |
|-----------------------------|-------------|-------------|
| красный соус, соус красный  | Красный соус | Красн_с/с   |
| сыр пармезан, пармезан      | Пармезан сыр | Пармезан   |
| сливочный соус              | Сливочный соус | Сливоч_с/с |
