# Формат этикетки DDLabel

Эталон макета этикетки 30×20 мм для печати на Xprinter XP-365B.

---

## Расположение и поля

| Элемент | Расположение | Содержимое |
|---------|--------------|------------|
| **Название продукта** | Вверху, по центру | Одна строка (при длинном названии — перенос или сокращение). Шрифт: крупный, жирный, с подчёркиванием. |
| **Дата начала** | Слева, под названием | ДД.ММ (дата изготовления). Крупнейший шрифт, жирный. |
| **Дата окончания** | Справа, на одной линии с датой начала | ДД.ММ (срок годности до). Тот же размер и начертание. |
| **Символ ∞** | По центру между датами и временем | Разделитель «от — до». |
| **Время начала** | Слева, под датой начала | ЧЧ.ММ (время изготовления). Крупный шрифт, жирный, с подчёркиванием. |
| **Время окончания** | Справа, под датой окончания | ЧЧ.ММ (время окончания срока). Тот же стиль. |

**Схема (две колонки):**

```
        Название продукта
              (подчёркнуто)

  30.01          ∞          02.02
  (дата          (символ)   (дата
   изг.)                     до)

  10.00                    23.59
  (время                   (время
   изг.,                    до,
   подчёркнуто)             подчёркнуто)
```

---

## Форматы данных

- **Даты:** ДД.ММ (год при необходимости — ДД.ММ.ГГГГ; по макету — кратко ДД.ММ).
- **Время:** ЧЧ.ММ (24 часа).
- **Символ:** ∞ (Unicode U+221E) или аналогичный разделитель «от — до».

---

## Стиль

- Чёрный текст на белом (термоэтикетка).
- Иерархия: даты — самый крупный размер; время — крупный; название — крупный, с подчёркиванием. Время снизу тоже с подчёркиванием.
- Выравнивание: название по центру; даты и время — две колонки (начало слева, окончание справа), символ ∞ по центру.

---

## Шрифты

- **Используем встроенные шрифты принтера (ESC/POS)** — без загрузки своих шрифтов.
- Конкретный **шрифт и начертание** (размер в точках: 9×17, 12×24, 24×24; жирность, подчёркивание) будут указаны пользователем позже. До уточнения — ориентир: 12×24 или 9×17 для этикетки 30×20 мм, 3–4 строки.
- В коде: выбор режима шрифта и начертания через ESC/POS-команды (размер, подчёркивание и т.д.).

---

## Реализация

- Печать через **встроенные шрифты ESC/POS**: размеры и начертание по командам принтера (конкретные значения — после уточнения пользователем). Позиционирование — пробелы/табуляция, символ ∞ — из набора символов принтера или замена при необходимости.
- Эталон визуала: см. приложенное изображение формата (сыр Российский, 30.01 / 02.02, 10.00 / 23.59, ∞ по центру).
