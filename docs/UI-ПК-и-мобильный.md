# Разделение отображения: ПК и мобильный

В проекте явно разделены стили для **ПК** и для **мобильного** (телефон/планшет). От этого зависит, куда вносить правки по виду.

## Breakpoint

- **ПК (desktop):** ширина экрана **от 1025px**
- **Мобильный:** ширина экрана **до 1024px**

## Где вносить правки

Файл: **`client/src/App.css`**

| Для чего | Где править |
|----------|-------------|
| **Только для ПК** (десктоп) | Блок с комментарием `/* ========== ПК (desktop): от 1025px ========== */` — правила внутри `@media (min-width: 1025px) { ... }` |
| **Только для мобильного** (телефон) | Блок с комментарием `/* ========== Мобильный (телефон/планшет): до 1024px ========== */` — правила внутри `@media (max-width: 1024px) { ... }` |

Общие стили (без медиа-запроса) применяются и на ПК, и на телефоне; при конфликте побеждают правила из соответствующего блока ПК или мобильного.
