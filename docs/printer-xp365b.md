## Подключение к телефону (Android)

- **Программа:** «Print Label» (установка из Google Play)
- Включить Bluetooth на телефоне, загрузить этикетки в принтер, включить принтер
- В приложении: **Setting** → **Connection method** → **Bluetooth** → выбрать принтер

### Печать из DDLabel по Bluetooth (п. 8.5)

DDLabel поддерживает печать с телефона через **Web Serial API** (Bluetooth SPP):

- **Требования:** Chrome 117+ (ПК) или Chrome 138+ (Android), HTTPS (или localhost)
- **Подготовка:** отключить USB от ПК, сопрячь принтер с телефоном в настройках Bluetooth
- **Сценарий:** при печати с телефона (Vercel) сервер возвращает 503 (USB недоступен) → приложение запрашивает TSPL через `/api/print-tspl` → пользователь выбирает принтер в диалоге → данные отправляются по Bluetooth SPP

## Характеристики (руководство + страница товара)

| Параметр | Значение |
|----------|----------|
| Метод печати | Прямая термопечать (картриджи/лента не нужны) |
| Разрешение | 203 dpi |
| Интерфейс | USB, Bluetooth |
| Ширина печати | 80 мм (регулируется 20–80 мм) |
| Скорость печати | мин 50,8 мм/с, макс 127 мм/с |
| Диаметр рулона | 20–80 мм |
| Питание | Внешний БП 24 В 2,5 А |
| Температура | +10…+40 °C |
| Обнаружение этикетки | Автоматически |

**Режимы:** печать этикеток/штрихкодов и печать чеков; переключение — при включении удерживать PAUSE, затем FEED.

**Рекомендуемые этикетки:** 75×120 мм, **не более 250 этикеток в рулоне** (рулоны от 300 шт. — риск замятия). Рекомендованные рулоны на pc-controllers.ru (артикулы): 44618, 44811, 45610, 45611, 45612, 45613, 45614. Для нашего сценария — этикетки 30×20 мм (ширина 30 мм входит в диапазон 20–80 мм).

**Важно (по данным продавца):** для «ручной» вёрстки этикеток продавец рекомендует BarTender; работа из Word/Excel не гарантируется. **Для DDLabel** печать будет из нашего приложения (Node.js, ESC/POS) по сформированному тексту; BarTender можно использовать опционально только для дизайна шаблонов. Ознакомительная версия BarTender (30 дней): https://ru.seagullscientific.com/label-software/barcode-label-design-and-printing/bartender-basic-edition/

## Кнопки и индикаторы

- **FEED** — подача бумаги; при переключении режимов — выбор режима
- **PAUSE** — пауза/продолжение; при включении с PAUSE — калибровка датчика зазора / печать текущего режима
- **ON LINE** (синий) — питание, готовность; мигает — пауза или печать
- **ERROR** (красный) — ошибка (крышка открыта, нет бумаги, перегрев и т.д.)

## Печать из DDLabel по USB (Windows): Zadig и WinUSB

Приложение DDLabel отправляет данные на принтер **напрямую по USB** (библиотека escpos-usb), а не через драйвер Windows. После установки стандартного драйвера (Seagull/Xprinter) Windows «занимает» устройство, и приложение не видит принтер («Принтер не найден»). Чтобы DDLabel мог печатать, нужно переключить USB-устройство принтера на драйвер **WinUSB**.

**Шаги (кратко):**

1. Скачать **Zadig**: https://zadig.akeo.ie/ (файл `zadig-x.x.exe`).
2. Подключить принтер по USB, включить его.
3. Запустить Zadig от имени администратора.
4. Меню **Options** → включить **List All Devices**.
5. В выпадающем списке выбрать устройство принтера (например, **Xprinter XP-365B** или **USB Printing Support**). Убедиться, что указаны совпадающие VID/PID (или название принтера).
6. В поле драйвера справа выбрать **WinUSB** (или **usbccgp** не использовать; нужен именно WinUSB).
7. Нажать **Replace Driver** (или **Install Driver**) и дождаться окончания.
8. Перезапустить сервер DDLabel (`npm start` в папке `server`) и снова нажать «Печать тестовой этикетки» в браузере.

**Важно:** после замены драйвера на WinUSB принтер может перестать появляться в «Устройства и принтеры» для обычной печати из Word и т.п. Для печати только из DDLabel этого достаточно. Если нужна и печать из других программ, можно вернуть стандартный драйвер через Диспетчер устройств (обновить драйвер → выбрать из списка → Xprinter).

## Устранение неполадок (кратко)

- Полосы на этикетке — проверить USB-кабель, перезапустить ПК и принтер
- Полосы/неполная печать — проверить печатающую головку, при необходимости протереть спиртом
- Замятие — уменьшить диаметр рулона в лотке
- Пустая этикетка — использовать именно **термоэтикетки**, не для термотрансферного принтера
