# DDLabel
Интеллектуальное приложение по составлению и печатанию этикеток со сроками.

## Запуск (Этап 1 — ПК-прототип)

1. **Сервер** (в одном терминале):
   ```powershell
   cd server
   npm start
   ```
   Сервер (API): http://localhost:3001

2. **Фронт** (во втором терминале):
   ```powershell
   cd client
   npm run dev
   ```
   Интерфейс: http://localhost:3000 (проксирует `/api` на сервер 3001).

3. В браузере нажать «Печать тестовой этикетки». При подключённом Xprinter XP-365B по USB этикетка печатается; без принтера сервер вернёт сообщение «Принтер не найден».
